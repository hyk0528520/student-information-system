系统信息

欢迎使用学生信息管理系统！本系统基于Spring Boot开发，包含完整的学生管理功能。

技术栈： Spring Boot 2.7 + MyBatis-Plus + MySQL 8.0 + Redis 7.0 + Sa-Token + Thymeleaf

主要功能：

  • 学生信息管理（增删改查）
  • 课程与成绩管理
  • 在线请假审批
  • 公告发布
  • 数据统计与分析
